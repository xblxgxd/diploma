# Архитектура интеграции ИИ-ассистента

## Обзор
В этом документе описана архитектура интеграции ИИ-ассистента в приложение велосервиса.

## Компоненты системы

### 1. Frontend Components
- **ChatInterface.jsx**: Компонент интерфейса чата с ИИ-ассистентом
- **ChatButton.jsx**: Плавающая кнопка вызова чата, отображается на всех страницах
- **chatApi.js**: API-функции для взаимодействия с бэкендом

### 2. Backend Components
- **chatRouter.js**: Маршруты API для ИИ-ассистента
- **chatController.js**: Обработка запросов к ИИ-ассистенту
- **AIService.js**: Сервис для взаимодействия с ИИ-провайдерами
- **aiConfig.js**: Конфигурация ИИ-сервисов

## Потоки данных

### 1. Отправка сообщения пользователем
```
Пользователь вводит сообщение → ChatInterface.jsx → chatApi.js → /api/chat/message → chatController.js → AIService.js → ИИ-провайдер → Ответ → chatController.js → ChatInterface.jsx
```

### 2. Получение подсказок
```
ChatInterface.jsx → chatApi.js → /api/chat/suggestions → chatController.js → Возвращение подсказок → ChatInterface.jsx
```

## ИИ-провайдеры

### Поддерживаемые провайдеры:
1. **OpenAI (GPT)** - через пакет openai
2. **Anthropic (Claude)** - через пакет @anthropic-ai/sdk
3. **Google (Gemini)** - через пакет @google/generative-ai
4. **Mock-режим** - для разработки и тестирования

### Выбор провайдера
Провайдер выбирается через переменную окружения `AI_PROVIDER`:
- `mock` - для разработки
- `openai` - для использования OpenAI
- `anthropic` - для использования Anthropic
- `google` - для использования Google

## Безопасность

### 1. Аутентификация
- Все запросы к ИИ-ассистенту требуют JWT-токена
- Проверка прав доступа к беседам

### 2. Валидация
- Проверка длины и содержания сообщений
- Фильтрация потенциально вредоносного контента

### 3. Защита от злоупотреблений
- Rate limiting для предотвращения спама
- Мониторинг подозрительной активности

## Конфигурация

### Переменные окружения
- `AI_PROVIDER` - выбранный провайдер ИИ
- `OPENAI_API_KEY` - API-ключ для OpenAI
- `OPENAI_MODEL` - используемая модель (по умолчанию gpt-3.5-turbo)
- `ANTHROPIC_API_KEY` - API-ключ для Anthropic
- `GOOGLE_GEMINI_API_KEY` - API-ключ для Google Gemini
